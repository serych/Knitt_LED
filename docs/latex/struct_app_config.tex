\doxysubsection{Detailed Description}
Application configuration and runtime state. 

Most fields are persisted using Preferences (see \doxylink{_app_config_8cpp}{App\+Config.\+cpp}). Some fields (like counters) are runtime-\/only unless you decide to persist them.

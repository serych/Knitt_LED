Detailed Description

KnittLED application entry point.

Initializes hardware (OLED, NeoPixels, buttons), Wi-Fi, file system, OTA, and web UI. Implements knitting logic: stepping rows, confirmation, carriage sensor handling, and warning blink. Row stepping wraps around and respects row counting direction (rowFromBottom).

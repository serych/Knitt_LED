Detailed Description

Web UI implementation (HTML + REST endpoints).

This file is intentionally self-contained: it embeds the HTML/JS UI and registers WebServer routes. Doxygen comments for public APIs are in WebUi.h; this file documents key internal helpers and endpoints.

◆ webuiBegin()

void webuiBegin ( WebUiDeps deps)

Register all routes for the web UI on the provided server.

Routes:

  • GET / : HTML UI
  • GET /api/files : JSON list of pattern file paths
  • GET /api/pattern : Load pattern (query param file)
  • POST /api/pattern : Save pattern (JSON body)
  • POST /api/delete : Delete file (JSON body)
  • POST /api/row : Step row (+1/-1) (JSON body)
  • POST /api/confirm : Confirm current row and optionally auto-advance
  • GET /api/state : Current state for polling
  • GET /api/config : Read config
  • POST /api/config : Update config
  • GET /download : Download a pattern file
  • POST /upload : Upload a pattern file
